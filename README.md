# Portfolio OS

A “Portfolio OS” — more than a portfolio site. This is a personal hub for browsing projects, jumping to frequently used tools, and showing off interactive 3D scenes in a clean dashboard-style UI.

**Repo:** https://github.com/Twinspeakers/portfolio_OS  
**GitHub Pages URL (expected):** https://twinspeakers.github.io/portfolio_OS/

---

## Highlights

- **Dashboard** home with quick links, featured projects, and status widgets
- **Projects** index with filters/search + MDX-backed project pages
- **Quick Links** page with categorized tools/resources
- **Lab** page for interactive **Three.js / react-three-fiber** scenes
- Styled with **Tailwind + shadcn/ui** (dashboard aesthetic + subtle gradients)

---

## Tech Stack

- **Next.js** (App Router) + **React** + **TypeScript**
- **Tailwind CSS** + **shadcn/ui**
- **Three.js** + **react-three-fiber** (+ `@react-three/drei`)
- **GSAP** (animation)
- **MDX** + **Contentlayer** (content)
- **Supabase** (backend; not yet implemented)
